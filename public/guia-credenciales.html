Credenciales de Instagram (Meta)

Los 4 datos que pide la pestaña Conexiones: IG User ID, Page ID, Page Access Token y Long-lived User Token. Son de tu propia cuenta — no requieren revisión de Meta.

⬇ Descargar esta guía (.md) ← Volver al panel

Requisitos previos (una sola vez)

  1. La cuenta de Instagram de tu marca debe ser Profesional (Business o Creator) y estar vinculada a una Página de Facebook de la marca (se configura en Meta Business Suite).
  2. Necesitas una app de Meta: developers.facebook.com/apps → Crear app → tipo Business. Agrégale los productos Instagram y Facebook Login for Business.
  3. Tu usuario debe ser Admin de la app, de la Página y de la cuenta de IG.

Paso 1 — Token corto (Graph API Explorer)

  1. Abre developers.facebook.com/tools/explorer
  2. Arriba a la derecha, en Meta App, selecciona tu app.
  3. En User or Page deja User Token.
  4. En Permissions agrega: instagram_business_basic, instagram_business_content_publish, pages_show_list, pages_read_engagement, pages_manage_posts, business_management
  5. Click Generate Access Token → inicia sesión y acepta.
  6. Copia el token (dura ~1–2 horas; con eso basta).

Paso 2 — Token largo (60 días)

Pega esto en el navegador reemplazando APP_ID, APP_SECRET (app de Meta → Settings → Basic) y TOKEN_CORTO:

https://graph.facebook.com/v21.0/oauth/access_token?grant_type=fb_exchange_token&client_id=APP_ID&client_secret=APP_SECRET&fb_exchange_token=TOKEN_CORTO

El access_token de la respuesta es tu Long-lived User Token.

Paso 3 — Página + Page Token (no expira)

https://graph.facebook.com/v21.0/me/accounts?access_token=LONG_LIVED_USER_TOKEN

Busca la Página de tu marca en la respuesta y copia id → Page ID y access_token → Page Access Token.

Paso 4 — IG User ID

https://graph.facebook.com/v21.0/PAGE_ID?fields=instagram_business_account&access_token=PAGE_TOKEN

Copia instagram_business_account.id → IG User ID.

Paso 5 — Pegar en el panel

Vuelve al panel → tu marca → pestaña Conexiones → pega los 4 valores y pulsa Probar conexión. Si sale ✅ con el nombre de tu cuenta, guarda los cambios y listo: la app ya puede publicar en ese Instagram.

💡 El Long-lived User Token dura 60 días. Si además configuras META_APP_ID y META_APP_SECRET en Vercel, la app lo refresca sola cada semana y nunca caduca.
